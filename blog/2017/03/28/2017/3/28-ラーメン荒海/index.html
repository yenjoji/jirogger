2017/3/28 ラーメン荒海

Mar 28, 2017 · 1 minute read · Comments

ラーメン　大盛　ヤサイ中盛　味濃いめ　アブラ多め 相変わらずのいい景色。 ヤサイはシャキッとした感じも残しつつ、ちゃんと火が通っていて、甘味がでてるいいやさい。 スープは、マイルドな魚介系のうまみたっぷりスープ。おいしい。 味濃いめにすると、多めのヤサイでも薄まらなくて良い塩梅。

yenjojiさん(@yenjoji)がシェアした投稿 - 2017 3月 27 5:41午前 PDT

麺はいつもの太麺。丁度いい茹で加減だけど、周りが麺固めを頼んでいたので、次回はわたしも固めにしてみてもいいかなぁと感じた。
